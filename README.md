# Laboratório de Programação de Jogos 🎮

Este repositório contém os códigos dos jogos desenvolvidos ao longo da disciplina de **Laboratório de Programação de Jogos**, do curso de Ciência da Computação da UFF (Universidade Federal Fluminense).

## 🛠️ Tecnologias Utilizadas

Todos os projetos deste repositório foram construídos em **Python** utilizando:
- **[Pygame](https://www.pygame.org/):** Biblioteca principal para desenvolvimento de jogos 2D.
- **[PPlay](http://www2.ic.uff.br/pplay/):** Uma biblioteca auxiliar desenvolvida pela própria universidade (UFF) com o objetivo de facilitar a manipulação de componentes do Pygame para os alunos.
  
Ao longo do semestre, os seguintes jogos estão sendo desenvolvidos:

* `Pong` - Primeira etapa: recriação do clássico Pong.
* `Space Invaders` - Segunda etapa: desenvolvimento do clássico Space Invaders.
* `Straw Hat Odyssey` - Projeto final da disciplina jogo autoral.
